Quick start
  1. Toggle On for any challenge that needs a live instance per team.
  2. Set the Docker Image (the image your challenge container runs) and Port.
  3. Pick a Runtime: Docker for local dev, kCTF for most prod challenges, Kata-FC for pwn/kernel/RCE.
  4. Click Save. Players will immediately see the Launch button on that challenge.

Challenges without instancing enabled are completely unaffected — no panel is shown to players.
